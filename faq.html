FAQ – Frequently Asked Questions

What is Privpost.online?
Privpost.online is a free tool for sending confidential, self-destructing notes and files. Notes and attachments are deleted after being read or after a set number of views/expiry time.
How does it work?
You create a note (optionally with a file and password), set expiry and view limits, and get a unique link. Once the recipient opens the link, the note is destroyed according to your settings.
What file types are allowed?
Images and .txt files up to 50MB are supported.
Is my data really deleted?
Yes. Notes and files are permanently deleted from our server after being read or expired. We do not keep backups of destroyed notes.
Is it anonymous?
No registration is required. Your IP is stored only for anti-spam purposes and is not shared.
Can I use a password?
Yes, you can set a password for your note. Only those with the password can read it.
Can I contact you?
Yes, see the Contact page.
Is it free?
Yes, Privpost.online is completely free to use.
