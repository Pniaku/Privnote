About Privpost.online – secure, self-destructing notes & file sharing

Privpost.online is a free tool for sending confidential, self-destructing notes and files. Notes and attachments are deleted after being read or after a set number of views/expiry time. No registration required.

Features

  • One-time, self-destructing notes and file sharing
  • Password protection, burn-after-x-views, expiry date
  • QR code for easy sharing
  • Modern, responsive design
  • Anti-spam protection
  • Admin panel with statistics

Security

  • Notes and files are deleted after being read or expired
  • Passwords are hashed, no plain storage
  • No tracking cookies, only session for admin
  • All uploaded images are automatically stripped of metadata (EXIF, ICC, etc.) for privacy
